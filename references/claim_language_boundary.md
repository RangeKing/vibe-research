# Claim Language Boundary

Use this reference before rewriting titles, abstracts, figure captions, cover letters, or Results claims for selective journals. The goal is to prevent rhetorical upgrades from outrunning what the metric actually measures.

For each central metric, define allowed and forbidden wording:

| metric | physical meaning | not meaning | allowed verbs | forbidden verbs | required qualifier |
|---|---|---|---|---|---|
| ordinal diagnostic rank | relative rank under the supplied diagnostic index | physical stability, permanence, or stock-loss probability | ranks higher, is classified higher, has higher diagnostic support | is more stable, is more permanent, loses less carbon | under the specified diagnostic index |
| positive sink probability | model fraction maintaining a positive sink function | carbon-stock loss probability | maintains positive sink function in X of Y models | avoids carbon loss, guarantees uptake | model fraction, scenario, period, variable |
| component-support validation | agreement between component signals and high-risk cells | observed loss validation | component support is higher in specified cells | validates observed loss | component, denominator, source-data table |
| fire-emission fractional coefficient | bounded pathway coefficient for a fire-emission component | total terrestrial stock-loss probability | contributes a bounded fire-emission pathway coefficient | estimates total stock loss | coefficient source, bound, sensitivity rule |

## Required Practice

- Claims using "shift", "reversal", "dominates", "ranks higher", "probability", "loss", "stability", "permanence", or "risk" must name the metric and denominator.
- If a metric is ordinal, diagnostic, proxy-based, bounded, or index-weighted, keep that qualifier in the strongest manuscript-facing location where the claim appears.
- If the strongest allowed wording becomes too weak for the target journal, the fix is new evidence, validation, uncertainty propagation, or figure redesign, not stronger verbs.
- Do not improve rhetorical force by dropping qualifiers during `/draft`, `/polish`, cover-letter work, or abstract tightening.
