## Comment triage
| Comment ID | Type | Severity | Can be fully answered? | Best move |
|---|---|---|---|---|
| R1 | novelty / mechanism / controls / stats / writing / framing | High / Medium / Low | Yes / Partly / No | revise / soften / clarify / defer |

## Best action per category
### If the reviewer is right
- Acknowledge clearly
- State what changed
- Point to revised text or figure when available

### If the reviewer is partly right
- Concede the valid part
- Narrow the claim
- Explain what remains supported

### If the reviewer overreaches
- Stay calm
- Correct the misread politely
- Use evidence, not attitude

### If no new experiment is available
- Do not bluff
- Reframe the claim
- Add limitation language
- Clarify the supported scope
